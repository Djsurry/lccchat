About

What it is, why we made it, and how to get it

What it is

LCC Chat is a product we are develping to allow easy communication between LCC students without having to use email. It uses email verification to protect users identities and make sure only LCC students can use it. Eventually it will hopefully turn into a fully-fledged desktop app, but these things take time. I hope to be coming out with a proof of concept simple UI in the coming weeks. I think this is will be a great new tool for LCC students for both educational and social needs.

"I started this project because of conversations I had had with my friends as well as a way to practice my software development skills. I really hope people enjoy and use this tool." --David Surry

Our Goal

The main goal of this project is to help students connect to each other without having to email back and forth which becomes tedious at some point. With this program we will hopefully remove the dependancy for email to connect with other LCC students whether it be for asking one person for homework or working together on a group project.

What we learned

Throughout the process, we learned and practiced many skills. Some of which are listed below
  • Server Management: To host the webserver, we use an external server. Although not as easy as using sqaure space or wix, it allowed us to understand how to host our own server and work as server admins.
  • Webpage Building: Most people just use wix or square space to make websites today. We though it was important to learn how to make our own using HTML, CSS and JavaScript.
  • Low Level Networking: The webserver uses TCP protocal sockets to communicate with the app, so we had to deal with all the encrytion, authentication, and bits/bytes that comes along with it.
  • Programming: We mainly used python to run the backend of the webserver, make the GUI of the app, and control the networking of the app. We also used several languages mentioned above to create our website. All in all we were exposed to a full range of what python offers as well as using other coding languages.
  • Marketing: Along with all the technical skills, we still have to get people to download the app. This is where marketing skills comes in.
  • Time Management: If all that seems like a lot, thats because it is. This requires us to plan and execute in an efficient manner.
  • Collaboration: We had to make sure that everyone in the group had a role and that we were all working diligently to meet our goals.
